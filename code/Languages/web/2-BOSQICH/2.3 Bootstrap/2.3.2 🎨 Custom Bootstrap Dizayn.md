## 🎨 Custom Bootstrap Dizayn

### 📌 Dars maqsadi

O‘quvchilar Bootstrap’dan foydalanib:

- Standart ko‘rinishdan chiqishni
    
- O‘ziga xos dizayn yaratishni
    
- Bootstrap + custom CSS ni to‘g‘ri uyg‘unlashtirishni  
    o‘rganadilar.
    

---

## 🧠 Nega Custom Bootstrap?

Bootstrap default holatda:

- Juda ko‘p joyda ishlatilgan
    
- Vizual jihatdan tanish
    
- Brendga moslash qiyin
    

Custom dizayn orqali:

- Unikal UI yaratiladi
    
- Brend ranglari qo‘llanadi
    
- Professional ko‘rinish hosil bo‘ladi
    

---

## ⚙️ To‘g‘ri ulash tartibi

Bootstrap’ni **doim birinchi**, custom CSS’ni **keyin** ulash kerak.

```html
<link href="bootstrap.min.css" rel="stylesheet">
<link href="custom.css" rel="stylesheet">
```

---

## 🎯 1. Ranglar (Theme colors) ni o‘zgartirish

### Custom CSS orqali

```css
:root {
  --bs-primary: [[1d4ed8]];
  --bs-secondary: [[64748b]];
  --bs-success: [[16a34a]];
}
```

Bootstrap komponentlari avtomatik yangilanadi:

```html
<button class="btn btn-primary">Primary</button>
```

---

## ✍️ 2. Typography sozlash

```css
body {
  font-family: 'Inter', sans-serif;
  font-size: 16px;
}

h1, h2, h3 {
  font-weight: 600;
}
```

📌 Google Fonts ishlatish tavsiya etiladi.

---

## 📦 3. Komponentlarni override qilish

### Button dizaynini o‘zgartirish

```css
.btn-primary {
  background-color: [[1d4ed8]];
  border-radius: 10px;
  padding: 12px 24px;
}
```

---

### Card dizaynini yangilash

```css
.card {
  border: none;
  border-radius: 16px;
  box-shadow: 0 10px 25px rgba(0,0,0,0.08);
}
```

---

## 📐 4. Layout’ni kuchaytirish (Flex + Utilities)

Bootstrap utility’larini CSS bilan birga ishlatish:

```html
<div class="d-flex align-items-center justify-content-between">
```

```css
.custom-section {
  min-height: 100vh;
}
```

---

## ✨ 5. Animatsiya va micro-interactions

```css
.btn {
  transition: all 0.3s ease;
}

.btn:hover {
  transform: translateY(-2px);
}
```
