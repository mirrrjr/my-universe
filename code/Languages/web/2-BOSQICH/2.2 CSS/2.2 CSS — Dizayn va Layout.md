### 📌 Dars maqsadi

O‘quvchilar HTML sahifalarga **professional dizayn**, **toza layout** va **responsive ko‘rinish** berishni CSS yordamida o‘rganadilar.

---

## 🧠 Nazariy qism

### CSS nima?

**CSS (Cascading Style Sheets)** — HTML elementlarning tashqi ko‘rinishini boshqarish uchun ishlatiladi.

CSS yordamida:

- Ranglar va shriftlar beriladi
    
- Elementlar joylashtiriladi
    
- Responsive dizayn yaratiladi
    
- Animatsiyalar qo‘shiladi
    

---

### CSS ulash usullari

1. **Inline CSS**
    
2. **Internal CSS**
    
3. **External CSS (tavsiya etiladi)**
    

```html
<link rel="stylesheet" href="style.css">
```

## 📦 Box Model

CSS’ning eng muhim tushunchasi:

- `content`
    
- `padding`
    
- `border`
    
- `margin`
    

```css
.card {
  padding: 20px;
  margin: 10px;
  border: 1px solid [[ccc]];
}
```

## 📱 Responsive dizayn

### Media query

```css
@media (max-width: 768px) {
  body {
    background-color: lightgray;
  }
}
```

- Mobile-first yondashuv
    
- Turli ekranlar uchun moslashuv

## ✨ CSS Animatsiyalar

### Transition

```css
button {
  transition: all 0.3s ease;
}
```

### Hover effect

```css
button:hover {
  background-color: black;
  color: white;
}
```
