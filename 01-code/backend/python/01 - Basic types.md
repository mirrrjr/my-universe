# 01 - Basic types
## Asosiy tushuncha

Python tilida 4 ta `basic` tip bor:
- string
- integer
- float
- boolean

## Misol
```python
# Strings
name_with_single_quotes = "boot.dev"  # not so good
name_with_double_quotes = "boot.dev"  # so good

# Numbers
x = 5  # positive integer
y = -5  # negative integer

# Float
x = 5.2
y = -5.2

# Booleans
is_tall = True
is_short = False
```
